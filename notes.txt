CSS - Cascading style sheet

1) Types of CSS
==> inline CSS
==> internal CSS
==> External CSS
inline > internal > External

2) CSS Syntax
selector{
    property: value;
}

3) Types of selector
==> Elements selector - (h1) (h2) (p)
==> Class selector - (.) - (Multiple) - no of times we can use the same class name
==> Id selector - (#) - (Unique) - Only one time we can use the same id name
==> Universal selector - (*) - Apply the style to the entire site 
==> Group selector - class or id selector contains different tags. we have to use same style means we go with Group selector

i) using flex - it will allgin column into row
ii) Flex direction - row, row-reverse, column, column-reverse
iii) Justify content (X) - Start, center, end, Space-evenly, Space-around, Space-Between
iv) align items (Y) (set the height then only align item will work) - start, center, end
v) flex-wrap - no-wrap, wrap, wrap-reverse
vi) gap - to be assign with values (px,%,rem,cm,m,vw,vh)
vii) Align content (Y) (set the height) (enable the flex-wrap) - Start, center, end, Space-evenly, Space-around, Space-Between
viii) Flex (on item) - flex:1 
viv) order - Change the visual order of the property

8) Position - Moveing one place to Another place
==> Static - default
==> Relative - one place to Another place
==> absolute - it will search the non-Static parent, if is there means it will act as same Relative, no means it will search one more(on one is non-Static means it will go to the root(body))
==> fixed - it will be fixed to the area
==> Sticky - using top:0 Praperty once reach the limit it will act as a same fixed
